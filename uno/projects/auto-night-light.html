Automatic Night Light using LDR

In this project, an LED automatically turns ON when it is dark and OFF when it is bright using an LDR sensor.

Automatic Night Light – Detailed Notes

1. How It Works

The LDR (Light Dependent Resistor) changes its resistance with light. At night (darkness), its resistance is very high, and Arduino detects a low voltage, turning the LED ON. In bright light, resistance is low, voltage increases, and the LED is turned OFF.

2. Circuit Connection

  • LDR one end → 5V
  • LDR other end → Analog pin A0 + one side of 10kΩ resistor
  • Other side of 10kΩ resistor → GND
  • LED Anode (+) → Digital pin 9 through 220Ω resistor
  • LED Cathode (–) → GND

3. Resistor Values

  • LDR Voltage Divider Resistor: 10kΩ (adjust based on light conditions)
  • LED Current Limiting Resistor: 220Ω (to protect LED)

4. Applications

  • Streetlights that turn ON at night automatically
  • Home night lamps
  • Garden lighting systems

5. Key Functions Used

  • analogRead() – Reads LDR value (0–1023)
  • digitalWrite() – Turns LED ON/OFF
  • pinMode() – Sets pin as INPUT or OUTPUT

int ldrPin = A0;     // LDR connected to analog pin A0
int ledPin = 9;      // LED connected to digital pin 9
int ldrValue = 0;

void setup() {
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);  // To monitor LDR values
}

void loop() {
  ldrValue = analogRead(ldrPin);   // Read LDR value
  Serial.println(ldrValue);        // Print for debugging

  if (ldrValue < 300) {            // Threshold value for darkness
    digitalWrite(ledPin, HIGH);    // Turn LED ON
  } else {
    digitalWrite(ledPin, LOW);     // Turn LED OFF
  }
  delay(200);  // Small delay
}
