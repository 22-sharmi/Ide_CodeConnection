Arduino UNO - DHT11 Sensor

Measure temperature and humidity using the DHT11 sensor and display readings on Serial Monitor.

DHT11 Sensor – Detailed Notes

1. What is a DHT11 Sensor?

The DHT11 is a low-cost digital sensor that measures both temperature and humidity. It is widely used in beginner electronics projects with Arduino, ESP32, and Raspberry Pi.

2. Working Principle

The sensor contains a thermistor for temperature measurement and a capacitive humidity sensor for humidity. An internal chip converts the readings into a digital signal.

3. Features

  • Temperature range: 0°C to 50°C
  • Humidity range: 20% to 90% RH
  • Accuracy: ±2°C (temperature), ±5% (humidity)
  • Power: 3V–5V
  • Output: Digital signal

4. Pin Connections

  • VCC → Arduino 5V
  • GND → Arduino GND
  • DATA → Digital Pin 2 (can be changed in code)

5. Applications

  • Weather stations
  • Home automation
  • Greenhouses
  • Industrial environment monitoring

6. Library Requirement

Install the DHT sensor library by Adafruit and Adafruit Unified Sensor library in Arduino IDE to use the sensor easily.

Circuit Connection:

  • VCC → Arduino 5V
  • GND → Arduino GND
  • DATA → Digital Pin 2

#include "DHT.h"

#define DHTPIN 2
#define DHTTYPE DHT11

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float hum = dht.readHumidity();
  float temp = dht.readTemperature();

  Serial.print("Humidity: ");
  Serial.print(hum);
  Serial.print("%  Temperature: ");
  Serial.print(temp);
  Serial.println("°C");

  delay(2000);
}
