Arduino UNO - Joystick with Serial Monitor

This project reads joystick X and Y axis values and prints them in the Serial Monitor.

Joystick – Detailed Notes

1. What is a Joystick?

A joystick is an input device that measures movement along two axes: X (horizontal) and Y (vertical). Most joysticks also have a built-in push button when pressed down.

2. How It Works with Arduino

The joystick has two potentiometers that output analog values (0 to 1023) depending on the stick’s position. The push button is a simple digital switch (LOW when pressed).

3. Circuit Connection

  • VCC → 5V on Arduino
  • GND → GND on Arduino
  • VRx (X-axis) → A0 on Arduino
  • VRy (Y-axis) → A1 on Arduino
  • SW (Button) → D2 on Arduino

4. Applications

  • Robot or motor control
  • Game controllers
  • Camera pan/tilt systems

5. Key Functions Used

  • analogRead() – Reads analog values (0-1023) from X and Y pins.
  • digitalRead() – Detects button press (HIGH/LOW).
  • Serial.print() – Sends joystick values to Serial Monitor.

const int xPin = A0;  // Joystick X-axis
const int yPin = A1;  // Joystick Y-axis
const int buttonPin = 2;  // Joystick button

void setup() {
  Serial.begin(9600);      // Start Serial communication
  pinMode(buttonPin, INPUT_PULLUP);  // Button is active LOW
}

void loop() {
  int xValue = analogRead(xPin);    // Read X-axis
  int yValue = analogRead(yPin);    // Read Y-axis
  int buttonState = digitalRead(buttonPin); // Read button

  Serial.print("X: ");
  Serial.print(xValue);
  Serial.print(" | Y: ");
  Serial.print(yValue);
  Serial.print(" | Button: ");
  Serial.println(buttonState == LOW ? "Pressed" : "Released");

  delay(200);  // Small delay for readability
}
