Arduino UNO - Sound Sensor

Learn how to use a sound sensor with Arduino UNO to detect claps or noises.

Sound Sensor – Detailed Notes

1. What is a Sound Sensor?

A sound sensor module detects sound vibrations in the air. It usually has a built-in microphone and amplifier, and outputs either an analog or digital signal.

2. How It Works

When the sensor detects sound above a set threshold, it outputs a HIGH or LOW signal depending on the module type.

3. Circuit Connection

  • VCC → 5V
  • GND → GND
  • OUT → Digital Pin 2

4. Applications

  • Clap-controlled lights
  • Security alarms
  • Sound level detection

// Sound Sensor Test - Serial Monitor
const int soundSensorPin = 2;

void setup() {
  pinMode(soundSensorPin, INPUT);
  Serial.begin(115200);
}

void loop() {
  int value = digitalRead(soundSensorPin);
  Serial.println(value);
  delay(100);
}
