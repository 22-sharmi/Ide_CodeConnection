Arduino UNO - Serial Toggle

Control LED and buzzer using Serial Monitor. Send '1' to turn ON, '0' to turn OFF.

Understanding the Serial Monitor

1. What is the Serial Monitor?

The Serial Monitor is a tool in the Arduino IDE that allows you to send and receive data between your computer and Arduino board using USB. It’s like a chat window between you and the Arduino.

2. How It Works

  • Arduino communicates over the USB cable using serial communication (TX for transmit, RX for receive).
  • Serial.begin(9600); starts communication at 9600 baud rate (speed).
  • Serial.read() reads incoming characters.
  • Serial.print() or Serial.println() sends data to the computer.

3. Why Use It?

  • Control Arduino from your computer.
  • Monitor sensor readings in real-time.
  • Debug and test Arduino programs easily.

4. In This Project

We type '1' or '0' in the Serial Monitor to control both the LED and buzzer. Arduino listens for incoming characters and changes the output pins accordingly.


int ledPin = 13;
int buzzer = 8;

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buzzer, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  if (Serial.available()) {
    char cmd = Serial.read();
    if (cmd == '1') {
      digitalWrite(ledPin, HIGH);
      digitalWrite(buzzer, HIGH);
    } else if (cmd == '0') {
      digitalWrite(ledPin, LOW);
      digitalWrite(buzzer, LOW);
    }
  }
}
