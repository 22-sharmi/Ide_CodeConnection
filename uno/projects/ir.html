Arduino UNO - IR Obstacle Sensor

Detect obstacles using an IR sensor and print the status to the Serial Monitor.

IR Obstacle Sensor – Detailed Notes

1. What is an IR Obstacle Sensor?

An IR (Infrared) obstacle sensor detects objects by emitting IR light and checking for reflections. If an object is close, the reflected light is detected by the sensor's photodiode or phototransistor.

2. Working Principle

The sensor has two main parts: an IR LED (transmitter) and a photodiode/phototransistor (receiver). When the IR light bounces back from an object, the receiver detects it and changes the output signal.

3. How It Works with Arduino

  1. VCC powers the sensor (5V).
  2. GND connects to Arduino ground.
  3. OUT pin sends a LOW signal when an obstacle is detected, HIGH otherwise.
  4. Arduino reads this digital signal and acts accordingly.

4. Applications

  • Line-following robots
  • Object detection in automation
  • Security systems
  • Proximity sensing for interactive devices

5. Advantages

  • Simple to use
  • Low cost
  • Works in various lighting conditions

6. Limitations

  • Performance affected by strong sunlight
  • Limited detection range (usually < 30cm)

Circuit Connection:

  • IR OUT to digital pin 7
  • VCC to 5V
  • GND to GND

int irPin = 7;

void setup() {
  pinMode(irPin, INPUT);
  Serial.begin(9600);
}

void loop() {
  int obstacle = digitalRead(irPin);
  if (obstacle == LOW) {
    Serial.println("Obstacle Detected!");
  } else {
    Serial.println("No Obstacle");
  }
  delay(500);
}
