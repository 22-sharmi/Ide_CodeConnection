Arduino UNO - LED Blink

This project blinks the built-in LED on pin 13 of the Arduino Uno.

LED Blink – Detailed Notes

1. What is an LED?

An LED (Light Emitting Diode) is a semiconductor device that emits light when current flows through it.

2. How It Works with Arduino

The Arduino can supply voltage to the LED through a digital output pin. Turning the pin HIGH lights it up, turning it LOW turns it off.

3. Circuit Connection

  • If using the built-in LED, connect to pin 13 (already wired on Uno boards).
  • If using an external LED:
    • Anode (+) to pin 13 through a 220Ω resistor
    • Cathode (–) to GND

4. Applications

  • Status indicators
  • Debugging signals in projects
  • Basic learning for Arduino timing and control

5. Key Functions Used

  • pinMode() – Configures pin as INPUT or OUTPUT.
  • digitalWrite() – Sets pin HIGH or LOW.
  • delay() – Pauses the program for a given time in milliseconds.

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
