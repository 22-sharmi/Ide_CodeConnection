Arduino UNO - Reaction Time Game (Push Button)

Test your reflexes using a simple push button connected to Arduino UNO!

Reaction Time Game – Detailed Notes

1. Project Overview

This project measures how quickly you can press a push button after an LED turns on. The reaction time (in milliseconds) is displayed on the Serial Monitor.

2. How It Works

  • The player waits while the Arduino delays for a random time (2–5 seconds).
  • When the LED lights up, the timer starts immediately using millis().
  • When the player presses the button, the Arduino calculates how long it took to react.

3. Circuit Connections

  • LED → Pin 13 (through a 220Ω resistor)
  • Push Button → Pin 2
  • GND → Other side of button
  • (Optional) 10kΩ resistor → Between Pin 2 and GND (for pull-down)

4. Key Functions Used

  • digitalRead() – Detects the button press.
  • digitalWrite() – Controls the LED.
  • millis() – Tracks elapsed time for reaction measurement.
  • random() – Creates a random delay for unpredictability.

5. Applications

  • Reflex and reaction speed testing
  • Fun challenge games
  • Learning basic timing and input concepts with Arduino

// Reaction Time Game Using Push Button and LED

const int ledPin = 13;      // LED connected to digital pin 13
const int buttonPin = 2;    // Push button connected to digital pin 2

unsigned long startTime;
unsigned long reactionTime;
bool waitingForPress = false;

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buttonPin, INPUT_PULLUP);  // Use internal pull-up resistor
  Serial.begin(9600);
  randomSeed(analogRead(A0)); // Seed random from analog noise
  Serial.println("=== Reaction Time Game ===");
  Serial.println("Wait for LED, then press the button!");
}

void loop() {
  delay(random(2000, 5000)); // Random wait between 2–5 seconds

  digitalWrite(ledPin, HIGH);   // LED ON
  startTime = millis();         // Start timing
  waitingForPress = true;

  while (waitingForPress) {
    if (digitalRead(buttonPin) == LOW) { // Button pressed
      reactionTime = millis() - startTime;
      Serial.print("Your reaction time: ");
      Serial.print(reactionTime);
      Serial.println(" ms");
      waitingForPress = false;
      digitalWrite(ledPin, LOW);  // Turn off LED
      delay(2000); // Pause before next round
    }
  }
}
