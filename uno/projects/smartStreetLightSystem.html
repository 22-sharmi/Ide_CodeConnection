Arduino UNO - Smart Street Light System

Automatically turns a street-light LED ON at night and increases brightness during rain using an LDR and Rain Drop sensor.

Smart Street Light System – Notes

1. Project Overview

This project uses an LDR to detect ambient light and a Rain Drop sensor to detect rain. The streetlight (LED) turns ON when it is dark; if rain is detected, the light increases brightness automatically to improve visibility.

2. Components Used

  • Arduino UNO
  • LDR (2-pin) + 10kΩ resistor (for voltage divider)
  • MH-RD Rain Drop Sensor Module (A0 output)
  • High-power LED or LED module (use appropriate resistor or driver)
  • Jumper wires & breadboard

3. Circuit Connections

  • LDR (2-pin voltage divider)
    One end of LDR → 5V
    Other end of LDR → A0
    Same A0 point → 10kΩ resistor → GND
    (This forms the voltage divider so Arduino can read a voltage at A0)
  • Rain Sensor Module
    A0 (Analog OUT) → A1 on Arduino
    VCC → 5V
    GND → GND
  • LED (Street Light)
    LED + → D9 (PWM pin) via suitable resistor or driver
    LED - → GND

4. How It Works

The Arduino continuously reads the LDR (A0) and the Rain sensor (A1). When the LDR reading falls below a configurable threshold, the system considers it "dark" and turns the LED ON at a base brightness. If the rain sensor reading indicates wet conditions (below its threshold), the Arduino increases the LED PWM duty cycle to boost brightness. During daytime the LED remains OFF.

5. Adjustable Parameters

  • darkThreshold — LDR ADC value below which it is considered dark (adjust per your location)
  • rainThreshold — Rain sensor ADC value below which rain is considered detected (adjust using sensor potentiometer)
  • baseBrightness — PWM value (0–255) when it is night but not raining

6. Key Functions Used

  • analogRead() — Reads LDR & rain analog values
  • analogWrite() — Sets LED PWM brightness
  • Serial.print() — Debug output to Serial Monitor

// Smart Street Light System with Rain Boost

int ldrPin = A0;      // LDR input
int rainPin = A1;     // Rain sensor analog output
int ledPin = 9;       // LED (PWM pin)

int darkThreshold  = 400;  // LDR value below which it's considered dark (adjust)
int rainThreshold  = 600;  // Rain sensor value below which rain is detected (adjust)
int baseBrightness = 150;  // PWM (0-255) when dark but no rain

void setup() {
  Serial.begin(9600);
  pinMode(ledPin, OUTPUT);
}

void loop() {
  int ldrValue  = analogRead(ldrPin);
  int rainValue = analogRead(rainPin);

  Serial.print("LDR: ");
  Serial.print(ldrValue);
  Serial.print("  |  Rain: ");
  Serial.println(rainValue);

  int brightness = 0;

  // Activate only when it's dark
  if (ldrValue < darkThreshold) {
    brightness = baseBrightness; // base night brightness

    // If rain detected, increase to max brightness
    if (rainValue < rainThreshold) {
      brightness = 255; // full brightness in rain
      Serial.println("Rain detected — boosting brightness to MAX");
    } else {
      Serial.println("Night mode — normal brightness");
    }
  } else {
    brightness = 0;  
    Serial.println("Daytime — light OFF");
  }

  // Apply PWM to LED
  analogWrite(ledPin, brightness);

  delay(500);
}
