Arduino UNO - Touch Sensor Serial

Displays “Touched” or “Not Touched” on Serial Monitor based on sensor input.

Touch Sensor – Detailed Notes

1. What is a Touch Sensor?

A touch sensor detects physical contact or proximity. A common type for Arduino is the TTP223 capacitive touch sensor.

2. How It Works

The sensor senses changes in capacitance when a human finger (or conductive object) approaches or touches its surface. It outputs a HIGH signal when touched, LOW otherwise.

3. How It Works with Arduino

  1. VCC to 5V (or 3.3V for some models)
  2. GND to Arduino GND
  3. OUT to digital pin 2 (in this example)

4. Applications

  • Touch-based switches
  • Interactive art projects
  • Control panels without mechanical buttons

5. Advantages

  • No moving parts – longer lifespan
  • Waterproof and dustproof
  • Simple wiring

6. Limitations

  • May trigger accidentally if close to conductive objects
  • Works best with direct finger touch

Circuit Connection:

  • OUT to digital pin 2
  • VCC to 5V
  • GND to GND

int touchPin = 2; // Connect touch sensor output to pin 2

void setup() {
  pinMode(touchPin, INPUT);
  Serial.begin(9600);
}

void loop() {
  int touchValue = digitalRead(touchPin);
  if (touchValue == HIGH) {
    Serial.println("Touched");
  } else {
    Serial.println("Not Touched");
  }
  delay(500);
}
