Smart Water Level Indicator

A water level monitoring project using Arduino UNO, LEDs, and a buzzer to indicate tank levels and overflow conditions.

Smart Water Level Indicator – Notes

1. How the System Works

The water level sensor outputs an analog value (0–1023) based on immersion depth. According to the reading, LEDs and the buzzer activate to alert water level status.

2. Components Used

  • Arduino UNO
  • Water Level Sensor
  • Red LED – High Level Warning
  • Green LED – Normal Level Indicator
  • Active Buzzer – Overflow Alert
  • 220Ω resistors for LEDs

3. Circuit Connection

  • Sensor VCC → 5V
  • Sensor GND → GND
  • Sensor OUT → A0
  • Green LED → Pin 7
  • Red LED → Pin 8
  • Buzzer → Pin 9

4. Project Behavior

  • Low Water: Green LED ON
  • Medium: All OFF
  • High: Red LED ON + Buzzer ON

int sensorPin = A0;      
int greenLED = 7;
int redLED = 8;
int buzzer = 9;

int value = 0;

void setup() {
  Serial.begin(9600);

  pinMode(greenLED, OUTPUT);
  pinMode(redLED, OUTPUT);
  pinMode(buzzer, OUTPUT);
}

void loop() {
  value = analogRead(sensorPin);

  Serial.print("Water Level: ");
  Serial.println(value);

  if (value < 200) {
    // Low water
    digitalWrite(greenLED, HIGH);
    digitalWrite(redLED, LOW);
    digitalWrite(buzzer, LOW);
    Serial.println("Status: LOW");
  } 
  else if (value < 700) {
    // Medium
    digitalWrite(greenLED, LOW);
    digitalWrite(redLED, LOW);
    digitalWrite(buzzer, LOW);
    Serial.println("Status: MEDIUM");
  } 
  else {
    // High water
    digitalWrite(greenLED, LOW);
    digitalWrite(redLED, HIGH);
    digitalWrite(buzzer, HIGH);
    Serial.println("Status: HIGH / OVERFLOW WARNING!");
  }

  Serial.println("-------------------------");
  delay(500);
}
