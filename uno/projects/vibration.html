Arduino UNO - Vibration Sensor

Detect vibrations using the SW-420 sensor and print to Serial Monitor.

Vibration Sensor – Detailed Notes

1. What is a Vibration Sensor?

A device that detects movement or shaking. The most common type used with Arduino is the SW-420 vibration sensor.

2. Working Principle

Inside the sensor, a small spring is loosely fixed. When vibration occurs, the spring touches or moves away from a contact point. This changes the circuit state, sending a HIGH or LOW signal to the Arduino.

3. How It Works with Arduino

  1. VCC powers the sensor (5V).
  2. GND connects to Arduino ground.
  3. DO pin sends HIGH when vibration is detected, LOW otherwise.
  4. Arduino reads this digital signal and reacts accordingly.

4. Applications

  • Earthquake detection alarms
  • Machine vibration monitoring
  • Theft detection for vehicles or safes
  • Industrial equipment maintenance alerts

5. Advantages

  • Simple to use
  • Low cost
  • Works with both digital and analog outputs

6. Limitations

  • Can be triggered by minor noise or movement
  • Requires sensitivity adjustment via onboard potentiometer for accurate readings

Circuit Connection:

  • Sensor OUT to digital pin 6
  • VCC to 5V
  • GND to GND

int vibrationPin = 6;

void setup() {
  pinMode(vibrationPin, INPUT);
  Serial.begin(9600);
}

void loop() {
  int vib = digitalRead(vibrationPin);
  if (vib == HIGH) {
    Serial.println("Vibration Detected!");
  } else {
    Serial.println("No Vibration");
  }
  delay(500);
}
