Arduino UNO - Rain Alert System

This project activates a buzzer and LED automatically when rain is detected using a rain sensor.

Rain Alert System – Notes

1. Project Overview

This system alerts you instantly when rain starts. The rain sensor detects moisture, and if rain is present, the Arduino turns ON a buzzer and LED as a warning.

2. How the Project Works

The rain sensor output becomes LOW when water is detected. The Arduino reads this signal from a digital pin and triggers an alert by powering the LED and buzzer.

3. Circuit Connection

  • Rain Sensor D0 → D2 on Arduino
  • LED (+) → D8 on Arduino (with 220Ω resistor)
  • Buzzer (+) → D9 on Arduino
  • All GND pins → GND on Arduino
  • Rain Sensor VCC → 5V

4. Applications

  • Rain alert alarm systems
  • Home safety and garden protection
  • Automatic covering/roof mechanisms

5. Key Functions Used

  • digitalRead() – Reads rain sensor digital output.
  • digitalWrite() – Controls LED and buzzer.
  • delay() – Adds a small timing gap.

const int rainPin = 2;   // Rain sensor digital pin
const int ledPin = 8;    // LED output
const int buzzerPin = 9; // Buzzer output

void setup() {
  pinMode(rainPin, INPUT);
  pinMode(ledPin, OUTPUT);
  pinMode(buzzerPin, OUTPUT);
}

void loop() {
  int rainDetected = digitalRead(rainPin);

  if (rainDetected == LOW) { // LOW = rain detected
    digitalWrite(ledPin, HIGH);
    digitalWrite(buzzerPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
    digitalWrite(buzzerPin, LOW);
  }

  delay(200);
}
