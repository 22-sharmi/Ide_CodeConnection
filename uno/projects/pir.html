Arduino UNO - PIR Motion Sensor

Detect motion using a PIR sensor and print the status to the Serial Monitor.

PIR Motion Sensor – Detailed Notes

1. What is a PIR Motion Sensor?

A PIR (Passive Infrared) sensor detects motion by sensing changes in infrared radiation (heat) emitted by living beings or objects.

2. Working Principle

The PIR sensor has two slots made of special IR-sensitive material. When a warm object (like a human) moves across its field of view, the change in infrared levels between the two slots is detected and triggers the output signal.

3. How It Works with Arduino

  1. VCC powers the sensor (5V).
  2. GND connects to Arduino ground.
  3. OUT pin sends HIGH when motion is detected, LOW otherwise.
  4. Arduino reads this digital signal and takes action accordingly.

4. Applications

  • Automatic lighting systems
  • Security alarms
  • Energy-saving devices
  • Human presence detection

5. Advantages

  • Low power consumption
  • Wide detection range
  • Easy to integrate with microcontrollers

6. Limitations

  • Cannot detect motion through glass
  • May trigger false alarms due to sudden temperature changes
  • Requires a short warm-up time before operation

Circuit Connection:

  • PIR OUT to digital pin 8
  • VCC to 5V
  • GND to GND

int pirPin = 8;

void setup() {
  pinMode(pirPin, INPUT);
  Serial.begin(9600);
}

void loop() {
  int motion = digitalRead(pirPin);
  if (motion == HIGH) {
    Serial.println("Motion Detected!");
  } else {
    Serial.println("No Motion");
  }
  delay(1000);
}
