Arduino UNO - Water Level Sensor

This project reads water level values using an analog water sensor and displays the readings on the Serial Monitor.

Water Level Sensor – Detailed Notes

1. What is a Water Level Sensor?

A water level sensor measures the depth or presence of water using exposed conductive traces. More water → higher analog value; less water → lower value.

2. How It Works with Arduino

The sensor outputs a variable voltage (0–5V) depending on immersion. Arduino reads this voltage using an analog input pin (A0).

3. Circuit Connection

  • Sensor VCC → 5V
  • Sensor GND → GND
  • Sensor Analog Output → A0

4. Applications

  • Water tank level monitoring
  • Automatic pump control
  • Drainage monitoring

5. Key Functions Used

  • analogRead() – Reads analog water level value (0–1023).
  • Serial.begin() – Starts serial communication.
  • Serial.println() – Prints water level readings.

int waterPin = A0;   // Water level sensor connected to A0
int waterValue = 0;

void setup() {
  Serial.begin(9600);  // Start Serial Monitor
}

void loop() {
  waterValue = analogRead(waterPin);  // Read sensor value

  Serial.print("Water Level: ");
  Serial.println(waterValue);

  if (waterValue < 200) {
    Serial.println("Status: Very Low / No Water");
  } else if (waterValue < 500) {
    Serial.println("Status: Low");
  } else if (waterValue < 800) {
    Serial.println("Status: Medium");
  } else {
    Serial.println("Status: HIGH");
  }

  Serial.println("-----------------------");
  delay(500);
}
