Arduino UNO - Buzzer

This project toggles a buzzer ON and OFF connected to pin 8 of the Uno.

Buzzer – Detailed Notes

1. What is a Buzzer?

A buzzer is an electronic device that produces sound when electricity flows through it. It is commonly used for alarms, alerts, and notifications in electronics projects.

2. Types of Buzzers

  • Active Buzzer – Produces sound as soon as power is applied.
  • Passive Buzzer – Needs a signal (frequency) to produce sound.

3. How It Works

When voltage is applied, the buzzer's internal piezoelectric element vibrates, producing sound. The vibration frequency determines the tone of the sound.

4. Pin Connections

  • + Pin → Arduino Digital Pin 8 (in this project)
  • - Pin → Arduino GND

5. Applications

  • Alarms and alerts
  • Game buzzers
  • Doorbell systems
  • Timer notifications

6. How It Works in This Project

  1. Buzzer is connected to Pin 8 of Arduino UNO.
  2. `digitalWrite(HIGH)` turns it ON, `digitalWrite(LOW)` turns it OFF.
  3. `delay(500)` makes it beep for 0.5 seconds ON and OFF repeatedly.

Circuit Connection:

  • + Pin of buzzer → Arduino Digital Pin 8
  • - Pin of buzzer → Arduino GND

int buzzer = 8;

void setup() {
  pinMode(buzzer, OUTPUT);
}

void loop() {
  digitalWrite(buzzer, HIGH);
  delay(500);
  digitalWrite(buzzer, LOW);
  delay(500);
}
