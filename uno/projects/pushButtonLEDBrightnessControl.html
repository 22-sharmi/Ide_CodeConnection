Arduino UNO - LED Brightness Control (One Push Button)

Control the LED brightness in multiple levels using just a single push button!

LED Brightness Control – Detailed Notes

1. Project Overview

This project uses a single push button to cycle an LED through multiple brightness levels. Each time the button is pressed, the LED increases its brightness until the maximum level, then returns back to OFF.

2. How It Works

  • The button is connected to a digital input pin.
  • Each press advances to the next brightness step.
  • The LED is connected to a PWM pin (Pin 9) so that brightness can be controlled using analogWrite().
  • Debouncing is used to avoid multiple accidental presses.

3. Circuit Connections

  • LED → Pin 9 (PWM) through a 220Ω resistor
  • LED GND → Arduino GND
  • Push Button → Pin 2
  • Other side of button → 5V
  • 10kΩ resistor → From Pin 2 to GND (pull-down)
  • (Alternatively, internal pull-up can be used)

4. Key Functions Used

  • analogWrite() – Controls LED brightness.
  • digitalRead() – Reads button press.
  • millis() – Used to debounce the button.

5. Applications

  • Learning PWM and brightness control
  • Creating dimmable lights
  • Interactive button-based control systems

// LED Brightness Control Using One Push Button

const int buttonPin = 2;     // Push button
const int ledPin = 9;        // LED on PWM pin

// Define brightness steps
int brightnessLevels[] = {0, 50, 120, 200, 255};
int levelCount = 5;
int currentLevel = 0;

bool lastButtonState = LOW;
unsigned long lastDebounceTime = 0;
unsigned long debounceDelay = 50;

void setup() {
  pinMode(buttonPin, INPUT);   // Using external pull-down
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  bool reading = digitalRead(buttonPin);

  // Debounce logic
  if (reading != lastButtonState) {
    lastDebounceTime = millis();
  }

  if ((millis() - lastDebounceTime) > debounceDelay) {
    if (reading == HIGH && lastButtonState == LOW) {
      currentLevel++;
      if (currentLevel >= levelCount) currentLevel = 0;

      analogWrite(ledPin, brightnessLevels[currentLevel]);

      Serial.print("Brightness Level: ");
      Serial.println(brightnessLevels[currentLevel]);
    }
  }

  lastButtonState = reading;
}
