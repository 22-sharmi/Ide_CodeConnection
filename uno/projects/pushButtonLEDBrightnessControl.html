Arduino UNO - LED Brightness Control (One Push Button)

Control the LED brightness in multiple levels using just a single push button!

LED Brightness Control – Detailed Notes

1. Project Overview

This project uses a single push button to cycle the LED through increasing brightness levels. After reaching full brightness, it returns to OFF.

2. How It Works

  • One button cycles through brightness levels.
  • The LED is connected to a PWM pin (Pin 9).
  • analogWrite() sets brightness for each level.
  • Debouncing prevents multiple unwanted button readings.
  • Internal pull-up resistor is used for the push button.

3. Circuit Connections

  • LED (with 220Ω resistor) → Pin 9 (PWM)
  • LED GND → Arduino GND
  • Push Button → Pin 2
  • Other side of button → GND
  • Using internal pull-up, so **no external resistor required**

4. Key Functions

  • digitalRead()
  • analogWrite()
  • millis() for debounce timing

5. Applications

  • Dimmable LED lamps
  • Brightness selectors for gadgets
  • Learning PWM and button debouncing

// LED Brightness Control Using One Push Button

const int buttonPin = 2;     // Push button
const int ledPin = 9;        // LED on PWM pin

// Brightness steps
int brightnessLevels[] = {0, 50, 120, 200, 255};
int levelCount = 5;
int currentLevel = 0;

bool lastButtonState = HIGH;  // HIGH because using INPUT_PULLUP
unsigned long lastDebounceTime = 0;
unsigned long debounceDelay = 50; // 50ms debounce

void setup() {
  pinMode(buttonPin, INPUT_PULLUP);   // Using internal pull-up resistor
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  bool reading = digitalRead(buttonPin);

  // Debounce logic
  if (reading != lastButtonState) {
    lastDebounceTime = millis();
  }

  if ((millis() - lastDebounceTime) > debounceDelay) {
    // Button pressed when reading LOW (because of pull-up)
    if (reading == LOW && lastButtonState == HIGH) {
      currentLevel++;
      if (currentLevel >= levelCount) currentLevel = 0;

      analogWrite(ledPin, brightnessLevels[currentLevel]);

      Serial.print("Brightness Level: ");
      Serial.println(brightnessLevels[currentLevel]);
    }
  }

  lastButtonState = reading;
}
