Arduino UNO - Ultrasonic Sensor

Measure distance using HC-SR04 with Arduino UNO.

Ultrasonic Sensor – Detailed Notes

1. What is an Ultrasonic Sensor?

An ultrasonic sensor measures distance by sending sound waves above human hearing range and measuring the time it takes for them to bounce back from an object.

2. How It Works

The HC-SR04 sensor sends a short sound pulse from the Trig pin and listens for its echo on the Echo pin. The Arduino measures the time taken and calculates the distance.

3. Circuit Connection

  • Trig → Digital Pin 9
  • Echo → Digital Pin 10
  • VCC → 5V
  • GND → GND

4. Applications

  • Obstacle avoidance robots
  • Distance measurement tools
  • Level sensing (water, bins, etc.)

5. Key Functions Used

  • digitalWrite() – Controls the Trig pin to send the pulse.
  • pulseIn() – Measures the time for the echo to return.
  • Formula: distance = (duration × 0.034) / 2 where 0.034 cm/µs is speed of sound.

#define trigPin 9
#define echoPin 10

long duration;
int distance;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  Serial.begin(9600);
}

void loop() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  duration = pulseIn(echoPin, HIGH);
  distance = duration * 0.034 / 2;

  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");

  delay(1000);
}
