Arduino UNO - Rain Sensor with Serial Monitor

This project reads analog rain intensity values from the MH-RD Rain Sensor module and displays them in the Serial Monitor.

Rain Sensor – Detailed Notes

1. What is a Rain Sensor?

The MH-RD Rain Sensor consists of two parts: a rain detection board and a control module. When raindrops fall on the detection plate, conductivity increases, and the analog output value drops. More rain → lower readings. Dry surface → higher readings.

2. How It Works with Arduino

The control module provides two outputs:

  • Analog Output (A0): Gives values 0–1023. Lower value = more rain.
  • Digital Output (D0): Gives HIGH/LOW depending on rain threshold set by the onboard potentiometer.

Arduino can read both using analogRead() and digitalRead().

3. Circuit Connection

  • A0 (Analog Out) → Arduino A0
  • D0 (Digital Out) → Arduino D2 (or any digital pin)
  • GND → Arduino GND
  • VCC → Arduino 5V

4. Applications

  • Rain detection
  • Weather monitoring
  • Automatic window/roof systems

5. Key Functions Used

  • analogRead() – Reads analog rain value (0–1023).
  • Serial.println() – Prints readings to the Serial Monitor.

int rainAnalog = A0;   // Rain sensor analog pin
int rainDigital = 2;   // Rain sensor digital pin (D0)

void setup() {
  Serial.begin(9600);
  pinMode(rainDigital, INPUT);  // Digital rain detection
}

void loop() {
  int analogValue = analogRead(rainAnalog);     // Read analog rain level
  int digitalValue = digitalRead(rainDigital);  // Read digital rain alert

  Serial.print("Analog: ");
  Serial.print(analogValue);

  Serial.print("  |  Digital: ");
  Serial.println(digitalValue == LOW ? "RAIN DETECTED" : "NO RAIN");

  delay(500);
}
