Arduino UNO - LED Brightness Control with Joystick

This project demonstrates controlling LED brightness with a joystick. The first code uses only the Y-axis, while the second adds X-axis control and a button.

Joystick LED Brightness – Notes

1. How It Works

The joystick gives analog values (0–1023). These values are mapped to PWM range (0–255) using map(), and analogWrite() adjusts the LED brightness.

2. Circuit Connection

  • Joystick VCC → 5V, GND → GND
  • VRx (X-axis) → A0
  • VRy (Y-axis) → A1
  • SW (Button) → Pin 2
  • LED X → Pin 9 (220Ω resistor)
  • LED Y → Pin 10 (220Ω resistor)

3. Applications

  • Brightness control for multiple LEDs
  • Joystick-controlled robotics or UI inputs
  • Gaming console projects

1. Single Axis Joystick Control

This code controls the brightness of a single LED using the joystick's Y-axis.


const int yPin = A1;   // Joystick Y-axis
const int ledPin = 9;  // LED pin (PWM)

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  int yValue = analogRead(yPin);          // Read joystick Y-axis
  int brightness = map(yValue, 0, 1023, 0, 255);  // Map to PWM range
  analogWrite(ledPin, brightness);        // Set LED brightness
  delay(10);
}

2. Dual Axis Joystick Control with Button

This code controls two LEDs with both X and Y joystick axes. The button turns both LEDs off when pressed.


int xPin = A0;
int yPin = A1;
int buttonPin = 2;
int ledX = 9;
int ledY = 10;

void setup() {
  pinMode(ledX, OUTPUT);
  pinMode(ledY, OUTPUT);
  pinMode(buttonPin, INPUT_PULLUP);  // Joystick button
  Serial.begin(9600);
}

void loop() {
  int xValue = analogRead(xPin);
  int yValue = analogRead(yPin);
  int buttonState = digitalRead(buttonPin);

  int ledXBrightness = map(xValue, 0, 1023, 0, 255);
  int ledYBrightness = map(yValue, 0, 1023, 0, 255);

  analogWrite(ledX, ledXBrightness);
  analogWrite(ledY, ledYBrightness);

  if (buttonState == LOW) {
    analogWrite(ledX, 0);
    analogWrite(ledY, 0);
  }

  delay(50);
}
