Arduino UNO - Buzzer Tone Control with Joystick

This project demonstrates controlling buzzer tones with a joystick. The first code varies tone continuously with the X-axis, while the second plays distinct notes based on joystick direction and includes a stop button.

Joystick Buzzer Control – Notes

1. How It Works

The joystick outputs analog values (0–1023) from its X and Y axes. These values are mapped to frequencies and played using the tone() function. The button is read digitally and stops the buzzer when pressed.

2. Circuit Connection

  • Joystick VCC → 5V, GND → GND
  • VRx (X-axis) → A0
  • VRy (Y-axis) → A1
  • SW (Button) → Pin 2
  • Buzzer → Pin 8 or 9 (depending on code)

3. Applications

  • Interactive sound control with a joystick
  • DIY musical instrument projects
  • Game or robot sound effects

1. Single Axis Joystick Tone Control

This code maps the joystick’s X-axis value to a frequency range and plays that tone on a buzzer, allowing smooth pitch control.


const int xPin = A0;     // Joystick X-axis
const int buzzerPin = 8; // Buzzer pin

void setup() {
  pinMode(buzzerPin, OUTPUT);
}

void loop() {
  int xValue = analogRead(xPin);          // Read joystick X-axis
  int frequency = map(xValue, 0, 1023, 200, 2000); // Map joystick value to frequency
  tone(buzzerPin, frequency);             // Play tone at the mapped frequency
  delay(10);
}

2. Dual Axis Joystick Notes with Button

This code plays different musical notes based on joystick direction. Pressing the button stops the buzzer immediately.


int xPin = A0; 
int yPin = A1;
int buttonPin = 2;
int buzzerPin = 9;

void setup() {
  pinMode(buttonPin, INPUT_PULLUP);
  pinMode(buzzerPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int xValue = analogRead(xPin);
  int yValue = analogRead(yPin);
  int buttonState = digitalRead(buttonPin);

  int frequency = 0;

  if (xValue < 400) {
    frequency = 262;  // C note
  } else if (xValue > 600) {
    frequency = 294;  // D note
  } else if (yValue < 400) {
    frequency = 330;  // E note
  } else if (yValue > 600) {
    frequency = 349;  // F note
  }

  if (frequency > 0 && buttonState == HIGH) {
    tone(buzzerPin, frequency);  // Play note
  } else {
    noTone(buzzerPin);          // Stop buzzer
  }

  if (buttonState == LOW) {
    noTone(buzzerPin);          // Stop if button is pressed
  }

  delay(50);
}
