Arduino UNO - Buzzer Tone Control with Joystick

This project changes buzzer tone frequency based on joystick X-axis position.

Joystick Buzzer Tone – Notes

1. How It Works

The joystick X-axis outputs values (0–1023), which we map to a frequency range. The tone() function generates a PWM signal at this frequency on the buzzer pin.

2. Circuit Connection

  • Joystick VCC → 5V, GND → GND
  • VRx (X-axis) → A0
  • Buzzer → Pin 8 (active buzzer recommended)

3. Applications

  • Joystick-based music control
  • Interactive sound effects
  • Learning frequency generation

const int xPin = A0;     // Joystick X-axis
const int buzzerPin = 8; // Buzzer pin

void setup() {
  pinMode(buzzerPin, OUTPUT);
}

void loop() {
  int xValue = analogRead(xPin);          // Read joystick X-axis
  int frequency = map(xValue, 0, 1023, 200, 2000); // Map to frequency range
  tone(buzzerPin, frequency);             // Play tone
  delay(10);
}
