Arduino UNO - Smart Weather Station

This project monitors temperature, humidity, light level, and rainfall using various sensors and displays the data on the Serial Monitor.

Smart Weather Station – Notes

1. What This Project Does

This Smart Weather Station reads multiple environmental values such as:

  • Temperature & Humidity (DHT11)
  • Light Intensity (LDR)
  • Rain Detection (Raindrop Sensor)
  • Alerts using LED + Buzzer

All data is printed to the Serial Monitor, and alerts are triggered when conditions exceed safe levels.

2. Components Used

  • Arduino UNO
  • DHT11 Temperature & Humidity Sensor
  • LDR (Light Dependent Resistor) + 10k Resistor
  • Raindrop Sensor Module
  • LED
  • Buzzer
  • Jumper Wires
  • Breadboard

3. Circuit Connections

DHT11 Sensor:

  • VCC → 5V
  • GND → GND
  • DATA → D2

LDR Sensor:

  • One end → 5V
    Other end → A0
    Same A0 point → 10kΩ resistor → GND
    (This creates the required voltage divider for correct LDR reading)

Rain Sensor Module:

  • A0 (Analog OUT) → A1 on Arduino
  • VCC → 5V
  • GND → GND

LED & Buzzer Alerts:

  • LED → D8 (via 220Ω resistor)
  • Buzzer → D9
  • Both GND → Arduino GND

4. Alert Conditions

  • High Temperature Alert: LED + buzzer activate when temperature > 30°C
  • High Humidity Alert: Alerts when humidity > 80%
  • Rain Alert: Alerts when rain value drops below 500 (wet condition)
  • Dark Environment Alert: Alerts when LDR value < 300

5. Key Functions Used

  • dht.readTemperature() – Reads temperature in °C
  • dht.readHumidity() – Reads humidity (%)
  • analogRead() – Reads LDR and Rain sensor values
  • digitalWrite() – Controls LED & buzzer alerts
  • Serial.print() – Displays weather data on Serial Monitor

#include <DHT.h>

#define DHTPIN 2
#define DHTTYPE DHT11
DHT dht(DHTPIN, DHTTYPE);

int ldrPin = A0;
int rainPin = A1;

int ledPin = 8;
int buzzerPin = 9;

void setup() {
  Serial.begin(9600);
  dht.begin();
  
  pinMode(ledPin, OUTPUT);
  pinMode(buzzerPin, OUTPUT);
}

void loop() {
  float temperature = dht.readTemperature();
  float humidity = dht.readHumidity();
  int lightValue = analogRead(ldrPin);
  int rainValue = analogRead(rainPin);

  Serial.println("----- Weather Station Data -----");

  Serial.print("Temperature: ");
  Serial.print(temperature);
  Serial.println(" °C");

  Serial.print("Humidity: ");
  Serial.print(humidity);
  Serial.println(" %");

  Serial.print("Light Level (LDR): ");
  Serial.println(lightValue);

  Serial.print("Rain Level: ");
  Serial.println(rainValue);

  // Alert conditions
  bool alert = false;

  if (temperature > 30) alert = true;
  if (humidity > 80) alert = true;
  if (rainValue < 500) alert = true;
  if (lightValue < 300) alert = true;

  if (alert) {
    digitalWrite(ledPin, HIGH);
    digitalWrite(buzzerPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
    digitalWrite(buzzerPin, LOW);
  }

  Serial.println("--------------------------------");
  delay(1000);
}
