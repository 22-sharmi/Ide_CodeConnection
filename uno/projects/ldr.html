Arduino UNO - LDR Sensor

This project reads values from an LDR (Light Dependent Resistor) using Arduino and displays them on the Serial Monitor.

LDR – Detailed Notes

1. What is an LDR?

An LDR (Light Dependent Resistor) is a resistor whose resistance changes with the intensity of light falling on it. More light → lower resistance, less light → higher resistance.

2. How It Works with Arduino

The LDR is usually connected in a voltage divider circuit with a fixed resistor. Arduino reads the changing voltage through its analog input pin (A0).

3. Circuit Connection

  • One side of the LDR → 5V
  • Other side of the LDR → Analog pin A0 and one end of a 10kΩ resistor
  • Other end of the resistor → GND

4. Applications

  • Automatic streetlights
  • Smart home lighting
  • Light intensity measurement

5. Key Functions Used

  • analogRead() – Reads analog value (0–1023) from the LDR pin.
  • Serial.begin() – Starts serial communication to display values.
  • Serial.println() – Prints LDR readings in the Serial Monitor.

int ldrPin = A0;  // LDR connected to analog pin A0
int ldrValue = 0;

void setup() {
  Serial.begin(9600);  // Start serial communication
}

void loop() {
  ldrValue = analogRead(ldrPin);  // Read the LDR value
  Serial.println(ldrValue);       // Print it to Serial Monitor
  delay(500);                     // Small delay for readability
}
