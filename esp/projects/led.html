ESP32 - LED Blink

This project blinks an LED connected to pin 2 of ESP32.

Circuit Connection:

  • LED +ve to GPIO 2
  • LED -ve to 220Ω resistor to GND

void setup() {
  pinMode(2, OUTPUT);
}
void loop() {
  digitalWrite(2, HIGH);
  delay(500);
  digitalWrite(2, LOW);
  delay(500);
}
