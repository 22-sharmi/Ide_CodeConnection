ESP32 - Buzzer

Buzz the buzzer every second using GPIO 15 of ESP32.

Circuit Connection:

  • Buzzer +ve to GPIO 15
  • Buzzer -ve to 220Ω resistor to GND

void setup() {
  pinMode(15, OUTPUT);
}
void loop() {
  digitalWrite(15, HIGH);
  delay(500);
  digitalWrite(15, LOW);
  delay(500);
}
