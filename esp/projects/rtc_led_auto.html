RTC-Based Home Automation LED Scheduler

Automatically turns an LED ON/OFF based on specific real-time clock values.

RTC LED Scheduler – Notes

1. What This Does

The LED turns ON at evening 6:00 PM (18:00) and turns OFF at morning 6:00 AM (06:00) automatically.

2. Real-Time Explanation

The RTC starts counting time from when the sketch is uploaded unless you set the time manually. If time looks wrong, you must adjust it using rtc.adjust().

3. Wiring

  • LED → GPIO 5
  • SDA → GPIO 21
  • SCL → GPIO 22

#include <Wire.h>
#include "RTClib.h"

RTC_DS3231 rtc;
int ledPin = 5;

void setup() {
  Serial.begin(115200);
  Wire.begin(21, 22);

  pinMode(ledPin, OUTPUT);

  if (!rtc.begin()) {
    Serial.println("RTC not found!");
    while (1);
  }
}

void loop() {
  DateTime now = rtc.now();
  int hour = now.hour();

  if (hour >= 18 || hour < 6) {
    digitalWrite(ledPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
  }

  delay(1000);
}
