RTC Time & Date Reader

Reads real-time clock values from Tiny RTC DS1307/DS3231 and displays them on the Serial Monitor.

RTC – Notes

1. What is RTC?

RTC stands for Real Time Clock. It is a small hardware clock module with its own battery, so it can keep track of:

  • Current time (hours, minutes, seconds)
  • Date (day, month, year)
  • Even when the ESP32 is powered OFF

Modules like DS1307 and DS3231 are commonly used. Among them, the DS3231 is more accurate

2. What This Does

This project reads the current date and time from the RTC module and prints it to the Serial Monitor every second.

3. Real-Time Behavior

Real time starts from the moment the sketch is uploaded because the RTC is set using the computer’s compile time unless you manually adjust the date/time.

4. Wiring

  • SDA → GPIO 21 (ESP32)
  • SCL → GPIO 22 (ESP32)
  • VCC → 3.3V or 5V
  • GND → GND

#include <Wire.h>
#include "RTClib.h"

RTC_DS1307 rtc;

void setup() {
  Serial.begin(115200);
  Wire.begin(21, 22);

  if (!rtc.begin()) {
    Serial.println("RTC not found!");
    while (1);
  }

  if (!rtc.isrunning()) {
    rtc.adjust(DateTime(F(__DATE__), F(__TIME__)));
  }
}

void loop() {
  DateTime now = rtc.now();

  Serial.print("Date: ");
  Serial.print(now.year());
  Serial.print("/");
  Serial.print(now.month());
  Serial.print("/");
  Serial.print(now.day());

  Serial.print("  Time: ");
  Serial.print(now.hour());
  Serial.print(":");
  Serial.print(now.minute());
  Serial.print(":");
  Serial.println(now.second());

  delay(1000);
}
