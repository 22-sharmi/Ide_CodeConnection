RTC Alarm Clock With Buzzer

Plays a buzzer alarm when the RTC time matches the preset alarm time.

RTC Alarm – Notes

1. What This Does

The buzzer rings when the clock time equals the alarm time you set inside the code.

2. Real-Time Behavior

RTC time begins from the moment the code is uploaded unless the user manually adjusts time. This is why alarms may ring earlier/later if not set manually.

3. Wiring

  • Buzzer → GPIO 9
  • SDA → GPIO 21
  • SCL → GPIO 22

#include <Wire.h>
#include "RTClib.h"

RTC_DS3231 rtc;

int buzzer = 9;
int alarmHour = 8;
int alarmMinute = 30;
int alarmSecond = 0;

void setup() {
  Serial.begin(115200);
  Wire.begin(21, 22);
  pinMode(buzzer, OUTPUT);

  if (!rtc.begin()) {
    Serial.println("RTC not found!");
    while (1);
  }
}

void loop() {
  DateTime now = rtc.now();

  if (now.hour() == alarmHour &&
      now.minute() == alarmMinute &&
      now.second() == alarmSecond) {

    digitalWrite(buzzer, HIGH);
    delay(5000);
    digitalWrite(buzzer, LOW);
  }

  delay(1000);
}
